<h1>ScrapBook 📸</h1>
<br>
A digital yearbook platform designed to help users organize, preserve, and relive their digital memories through curated interactive galleries and custom themes.

<h1>
🚀 Overview
</h1>
<br>
ScrapBook is a full-stack application built using the PERN stack (PostgreSQL, Express, React, Node.js). It enables users to catalog "Memories" (high-level collections) and "Moments" (specific photos and stories within those memories). The standout feature is the Yearbook Mode, which dynamically compiles a year's worth of data into a visually stunning, themed digital book.

<h1>
✨ Key Features
</h1>
<br>
<ul>
  <li>
      Themed Yearbooks: Automatically generates a curated yearbook for any given year with multiple layout templates, including Polaroid, Filmstrip, Neon Board, and Classic.
    
  </li>
  <li>
      Infinite Scrolling: High-performance browsing in memory and moment galleries using a custom Intersection Observer implementation.
  </li>
  <li>
      Secure Authentication: Leveraging Clerk for user authentication, ensuring private and secure access to personal data.
  </li
    
  <li>
      Robust Data Management: Built with PostgreSQL to handle relational data efficiently, including optimized join queries and database transactions to ensure data integrity.
    
  </li>
  
  <li>
      Premium UI/UX: A modern, dark-themed interface built with Tailwind CSS v4 and smooth animations powered by Framer Motion.
  </li>
  
</ul>


<h1>
🛠️ Tech Stack
  
</h1>
<br>
<ul>
  <li>
    Frontend: React 19, Tailwind CSS v4, Framer Motion, React Router v7.
  </li>
  <li>
    Backend: Node.js, Express v5, Clerk Express SDK.
  </li>
  <li>
    Database: PostgreSQL (Neon), pg (node-postgres) with Connection Pooling.
  </li>
  <li>
    Authentication: Clerk.
  </li>
  
</ul>
